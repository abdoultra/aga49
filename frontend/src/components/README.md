# Components

Ce dossier contient les elements reutilisables de l'interface.

Exemples : `Logo`, `PublicHeader`, `PublicFooter`, `Pagination`.

Un composant doit etre place ici lorsqu'il est utilise par plusieurs pages ou
qu'il represente un element autonome de l'interface.

## Navigation publique

`PublicHeader` contient la navigation principale du site public. L'acces a
l'administration reste dans le footer pour rester disponible sans ajouter une
action supplementaire dans l'en-tete public.

`PublicFooter` regroupe les liens utiles, les ressources, les coordonnees de
l'association et la ligne legale. Le credit `Site fait par Abdoultra` est place
dans la ligne basse afin de rester visible sur tout le site sans perturber le
contenu principal.
