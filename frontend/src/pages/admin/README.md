# Pages admin

Les premiers écrans privés sont maintenant présents :

- `LoginPage` : formulaire de connexion JWT ;
- `DashboardPage` : statistiques et données récentes ;
- `MembersPage` : CRUD complet des membres ;
- `MembershipFeesPage` : CRUD et suivi financier des cotisations ;
- `PublicationsPage` : CRUD partagé des actualités, annonces et événements ;
- `GalleryPage` : gestion des albums et des photos ;
- `DocumentsPage` : upload, publication et téléchargement des documents ;
- `MessagesPage` : boîte de réception et suivi des demandes publiques ;
- `SettingsPage` : profil, mot de passe et gestion des administrateurs ;
- `AdminPlaceholderPage` : attente des prochains CRUD.

Les principaux modules CRUD prévus par la maquette sont maintenant présents.
